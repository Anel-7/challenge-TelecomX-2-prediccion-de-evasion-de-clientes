📡 Desafío Telecom X - Parte 2 - Predicción de Evasión de Clientes

🌟 Descripción del Proyecto:

Telecom X enfrenta una alta tasa de cancelaciones y necesita predecir la evasión de clientes a partir de las características  
de su comportamiento y servicios contratados. En este proyecto se transforman los datos y se entrenan diversos modelos  
de Machine Learning para identificar a los clientes con mayor probabilidad de cancelar el servicio.

🎯 Objetivos:

  🧹 Preparar los datos para el modelado (tratamiento, codificación, normalización).
  🔍 Realizar análisis de correlación y selección de variables.
  🤖 Entrenar dos o más modelos de clasificación.
  📊 Evaluar el rendimiento de los modelos con métricas.
  📈 Interpretar los resultados, incluyendo la importancia de las variables.
  🎯 Crear una conclusión estratégica señalando los principales factores que influyen en la cancelación.

🏁 Pasos a seguir:

  📌 Extracción de datos:  
      Descargar los datos limpios desde el proyecto Telecom X parte 1.  
    
  🛠️ Pre procesamiento:  
     🔤 Encoding  
     🗑️ Eliminación de columnas irrelevantes (numéricas y categóricas)  

  ➗ Separación de datos
      Dividir entre datos de entrenamiento y datos de prueba  

  🏋️‍♂️ Entrenamiento y 🧐 Evaluación
      Modelo Regresión Logística  
      Modelo Regresión Logística balanceado  
      Modelo de Árbol de Decisión  
      Modelo Random Forest  
      Modelo KNN

  🐍 Python 3.12.12

  📊 Matplotlib – visualización de datos.
